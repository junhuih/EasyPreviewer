# 项目概览

这是一个只用于在线预览的文件预览项目，目标是提供适合商业使用的开源许可方案，并对捆绑第三方资源做明确声明，同时完整支持中英文双语。

## 核心原则

- 仅预览，不做编辑
- 仅支持中文和英文
- 使用 Apache-2.0 许可证
- 捆绑的第三方运行时资源继续沿用其原始许可证
- Office 预览以 LibreOffice 为核心引擎
- 功能范围清晰，明确列出不支持的能力

## V1 支持范围

- 通过 LibreOffice 转 PDF 的 Office 文档预览
- 通过捆绑表格预览栈实现 `.xlsx` 只读预览
- PDF 预览
- 文本和代码文件预览
- Markdown 预览
- 常见图片预览
- CSV 只读预览

## 明确不支持

- CAD 预览
- 基于 iText 5 的 TIFF 转 PDF
- 音视频转码
- 上传、删除、导出、保存回源文件、在线编辑
