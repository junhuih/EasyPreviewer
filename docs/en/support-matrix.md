# Support Matrix

| Category | Status | Notes |
| --- | --- | --- |
| Office to PDF (`doc`, `docx`, `ppt`, `pptx`, `odt`, `odp`, `wps`) | Supported | Converted by LibreOffice and rendered read-only |
| Spreadsheet viewer (`xls`, `xlsx`, `xlsm`, `xlt`, `xltm`, `ods`) | Supported | Spreadsheet preview path; `.xlsx` and `.xlsm` use the browser workbook viewer |
| PDF | Supported | Rendered read-only |
| Images (`png`, `jpg`, `jpeg`, `gif`, `webp`, `svg`) | Supported | Browser-native read-only rendering |
| Video (`mp4`, `webm`, `mov`, `m4v`, `ogg`) | Supported | Browser-native inline playback |
| Text / Code (`txt`, `json`, `xml`, `yaml`, `yml`, `java`, `js`, `ts`, `tsx`, `jsx`, `py`, `css`, `html`) | Supported | Read-only text/code viewer |
| Markdown (`md`, `markdown`) | Supported | Sanitized read-only markdown rendering |
| CSV | Supported | Native read-only table preview without LibreOffice |
| CAD (`dwg`, `dxf`, `dwf`, `ifc`) | Unsupported | Removed to avoid Aspose.CAD dependency |
| TIFF advanced conversion (`tif`, `tiff`) | Unsupported | iText 5 based conversion intentionally excluded |
| Non-browser-native media (`avi`, `rm`, `wmv`, `mkv`, `flv`, `3gp`) | Unsupported | Server transcoding intentionally excluded |
| Upload / Delete / Export / Edit | Unsupported | Product policy: preview only |
