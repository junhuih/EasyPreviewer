# Support Matrix

| Category | Status | Notes |
| --- | --- | --- |
| Office (`doc`, `docx`, `xls`, `xlsx`, `ppt`, `pptx`, `odt`, `ods`, `odp`, `wps`) | Supported | Converted by LibreOffice to previewable PDF |
| PDF | Supported | Rendered read-only |
| Images (`png`, `jpg`, `jpeg`, `gif`, `webp`, `svg`) | Supported | Browser-native read-only rendering |
| Text / Code (`txt`, `json`, `xml`, `yaml`, `yml`, `java`, `js`, `ts`, `py`, `css`, `html`) | Supported | Read-only text/code viewer |
| Markdown (`md`) | Supported | Sanitized read-only markdown rendering |
| CSV | Supported | Read-only text/table rendering |
| CAD (`dwg`, `dxf`, `dwf`, `ifc`) | Unsupported | Removed to avoid Aspose.CAD dependency |
| TIFF advanced conversion (`tif`, `tiff`) | Unsupported | iText 5 based conversion intentionally excluded |
| Non-browser-native media (`avi`, `rm`, `wmv`, `mkv`, `flv`) | Unsupported | Server transcoding intentionally excluded |
| Upload / Delete / Export / Edit | Unsupported | Product policy: preview only |
