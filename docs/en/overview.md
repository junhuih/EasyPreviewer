# Overview

This project is a preview-only file viewer for bilingual products that need a commercial-friendly open source license posture with explicit third-party notice handling.

## Principles

- Preview only
- Chinese and English only
- Apache-2.0 licensed
- Bundled third-party runtime assets remain under their own licenses
- LibreOffice-powered Office conversion
- Narrow scope with explicit unsupported capabilities

## Supported

- Office documents converted to PDF through LibreOffice
- `.xlsx` preview through a bundled spreadsheet viewer stack
- PDFs
- Plain text and code files
- Markdown files
- Common images
- CSV as read-only content

## Not Supported

- CAD preview
- TIFF to PDF based on iText 5
- Media transcoding
- Upload, delete, export, save-back, or editing features
